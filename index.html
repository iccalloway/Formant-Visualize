Smoothing

Adjusts the number of previous frames each measure is averaged against.

With the default value (2), each point is a weighted average of the current frame and the previous frame.

LPC Order

LPC Analysis attempts to compute an appropriate all-pole filter for the signal.

This parameter adjusts the number of poles it searches for.

Sampling Rate

Number of samples taken by your audio input per second.
